# Experience Book

Save your XP points in a book.

Recipe in the gallery.

Version: `1.21.4`

## Commands

Give:

```mcfunction
/loot give @s loot experience_book:experience_book
```

Disable:

```mcfunction
/datapack disable "file/experience-book.zip"
```

Enable:

```mcfunction
/datapack enable "file/experience-book.zip"
```
